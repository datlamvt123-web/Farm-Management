# Farm-Management
QUẢN LÝ CHĂN NUÔI TRONG TRANG TRẠI
* Ứng dụng chính SQL Server 2020
* Cách chạy 
 - Tải file Farm_management và file TruyVan
 - Mở ứng dụng làm bài và sao chép file Farm_management và dán vào Query --> Ấn chạy để create table (nếu các bạn có làm bằng ứng dụng khác như MySql,... thì phải thay đổi 1 chút code để có thể chạy)
 - Mở file TruyVan --> copy các dòng truy vấn vào Query vừa rồi và chạy từng cái để xem chi tiết
